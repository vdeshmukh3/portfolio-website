Cafe Zone App

The Cafe Zone App Dashboard project aims to create a comprehensive and user-friendly administrative dashboard for managing a café business. Developed using Android Studio and integrated with Firebase, this dashboard will enable café owners and managers to efficiently manage various aspects of their operations, including menu management, order tracking, Two waitlist System, and Live Restaurent Feed.
